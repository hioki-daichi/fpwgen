# password-generator

## How to develop

1. **Backend**
    ```shell
    $ git clone https://github.com/hioki-daichi/password-generator-api
    $ cd password-generator-api
    $ go run main.go
    ```
1. **Frontend**
    ```shell
    $ git clone https://github.com/hioki-daichi/password-generator
    $ cd password-generator
    $ npm start
    ```
